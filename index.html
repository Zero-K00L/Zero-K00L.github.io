About This Portfolio

My focus is secure full-stack software development. Throughout the Computer Science program, I have built skills in object-oriented programming, mobile development, database design, algorithms, testing, security, and technical communication. This portfolio shows how I applied those skills to improve earlier coursework into more professional artifacts.

The artifacts included here were selected because they represent different areas of computer science. Enhancement One

Software Design and Engineering

CS 360 Android Inventory App

This Android mobile application was enhanced to improve software structure, security, validation, and maintainability. The improved version includes salted password hashing, user-specific inventory records, database migration logic, stronger input validation, centralized adapter listener logic, and shared SMS helper behavior.

  • Improved authentication security
  • Separated inventory by logged-in user
  • Refactored repeated SMS and listener logic
  • Added stronger validation and defensive checks

Algorithms and Data Structures

This Java console application was enhanced to better demonstrate algorithmic thinking and data structure use. The improved version uses a centralized AnimalInventory class, HashMap-based lookup, set-based validation, reusable input helpers, filtering, sorting, and improved reservation logic.

Enhancement Three

Databases

CS 340 Grazioso Salvare MongoDB Dashboard

This Python, MongoDB, and Dash dashboard was enhanced to improve database access, data security, query efficiency, and dashboard reliability. The improved version removes hard-coded credentials, uses environment variables, pushes filtering into MongoDB queries, adds projection support, uses aggregation for breed counts, and includes safer map handling.

  • Removed hard-coded database credentials
  • Moved dashboard filtering into MongoDB queries
  • Added projection and aggregation support
  • Improved validation, indexing, and error handling
